# Amazon Clone Database

## Project Overview

This project is a MySQL database designed for an Amazon-like E-Commerce application. It demonstrates database design, table relationships, SQL queries, stored procedures, functions, triggers, and transactions.

## Technologies Used

* MySQL
* MySQL Workbench
* Git
* GitHub

## Database Tables

* Users
* Sellers
* Categories
* Products
* Inventory
* Addresses
* Cart
* Cart Items
* Wishlist
* Orders
* Order Items
* Payments
* Shipping
* Reviews
* Coupons
* Returns

## Features

* Database Design using Primary Keys and Foreign Keys
* CRUD Operations
* Joins
* Aggregate Functions
* GROUP BY and HAVING
* Subqueries
* Views
* Stored Procedures
* Functions
* Triggers
* Transactions

## Project Structure

* `01_database.sql` – Database schema
* `02_Sample_Data.sql` – Sample data
* `03.Sample_Queries.sql` – SQL practice queries
* `Procedure.sql` – Stored procedures
* `functions.sql` – User-defined functions
* `Triggers.sql` – Database triggers

## How to Run

1. Create a MySQL database.
2. Execute `01_database.sql`.
3. Execute `02_Sample_Data.sql`.
4. Run the queries in `03.Sample_Queries.sql`.
5. Execute procedures, functions, and triggers as required.
